## Deployment (Netlify)

This site is configured for Netlify with:
- **`_redirects`** вҖ” clean URL support (no `.html` in URLs)
- **`sitemap.xml`** вҖ” submitted to Google Search Console
- **`robots.txt`** вҖ” Sitemap URL included

### Deploy steps:
1. Push to GitHub
2. Connect repo to Netlify
3. Build command: *(none вҖ” static HTML)*
4. Publish directory: `.` (root)
